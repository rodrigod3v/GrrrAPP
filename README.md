# GrrrAPP

Welcome to the GrrrAPP project! This repository contains the source code for our amazing application.

## Prerequisites

- Node.js (v18+)
- npm or yarn

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rodrigod3v/GrrrAPP.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
